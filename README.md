# Computer Vision Final Project
DIAGNOSIS OF ALZHEIMER’S DISEASE USING BINARY CLASSIFICATION CNN OF MRI SCANS

Early diagnosis is the primary motivator for this project, as manual MRI analysis is time-consuming and prone to human subjectivity. We utilized a Convolutional Neural Network (CNN) to automate this process. This provides an objective, scalable diagnostic tool for high-volume neuroimaging data. The project focuses on binary classification between AD and MCI from images obtained from the ADNI (Alzheimer’s Disease Neuroimaging Initiative) dataset. We have three different models and have initialized hyper-parameter based on our referenced article. The project aims to compare results of these 3 models using different criteria.


Reference


AbdulAzeem, Y., Bahgat, W. M., & Badawy, M. (2021) . A CNN based framework for classification of Alzheimer's disease. Neural Computing and Applications . https://doi.org/10.1007/s00521-021-05799-w
